# Level 14: Conditional Expectation on σ-algebras

## Reading

In Level 13 we saw that conditioning on an event gives a number, and that it's more natural to think of conditional expectation as a *random variable* — a function that captures "our best guess given partial information." But we ran into trouble with continuous random variables, where events like $\{Y = y\}$ have probability zero.

The solution is to define conditional expectation axiomatically: we specify what properties it *must* have, and then show such an object exists and is essentially unique.

### The formal definition

**Definition 4 (Conditional expectation on a random variable or $\sigma$-algebra).** Let $(\Omega, \mathcal{F}, \mathbb{P})$ be a probability space. Let $X$ be a real-valued random variable on $\Omega$, and $Y$ be an $(S, \mathcal{S})$-valued random variable on $\Omega$. Suppose that $\int_\Omega |X| \, \mathrm{d}\mathbb{P} < \infty$. Let $\mathcal{G} = \sigma(Y)$.

Suppose that $Z$ is a real-valued random variable satisfying:

**(i)** $\int_\Omega |Z| \, \mathrm{d}\mathbb{P} < \infty$,

**(ii)** $Z$ is $\mathcal{G}$-measurable, and

**(iii)** For all $A \in \mathcal{G}$ we have $\mathbb{E}(Z \mathbf{1}_A) = \mathbb{E}(X \mathbf{1}_A)$.

Then we say $Z$ is a **version** of $\mathbb{E}(X | Y)$; we write "$Z = \mathbb{E}(X | Y)$ almost surely." We may also (equivalently) write $\mathbb{E}(X | \mathcal{G})$.

### Understanding the three conditions

Let us unpack each condition:

**Condition (i)** is our "good behaviour" requirement: we don't want to deal with infinities.

**Condition (ii)** is the "sanity check": intuitively, it's telling us we aren't allowed to make different guesses for the value of $X$ at $\omega, \omega' \in \Omega$ if the value of $Y$ at those two points is the same. The conditional expectation can only depend on the information available through $Y$.

**Condition (iii)** is "we can integrate and get the right answer": by definition of the expectation, integrating is exactly what we're doing. This says that on any event in $\mathcal{G}$, the integral of our guess $Z$ equals the integral of the true variable $X$.

### Why condition on a $\sigma$-algebra?

Observe that $Y$ doesn't actually play a rôle in the definition except to define $\mathcal{G} = \sigma(Y)$. This justifies writing $\mathbb{E}(X | \mathcal{G})$. There is no loss of generality in assuming that $\mathcal{G}$ is generated by a random variable: for any $\mathcal{G} \subseteq \mathcal{F}$ a $\sigma$-algebra, there is a (not necessarily real-valued) random variable $Y$ with $\sigma(Y) = \mathcal{G}$.

While in simple examples we usually want to think of conditioning on a random variable (we are told the value of $Y$: what is now our best guess for $X$?), in more complicated settings it's notationally easier to condition on a $\sigma$-algebra (remember this too is conveying information!), and so this is what we'll usually do.

### Why integrate over $\mathcal{G}$-measurable events?

Integrating only over $\Omega$ is not restrictive enough: it would allow us to just take $Z = \mathbb{E}(X)$ to be a constant function. But we "should" be taking into account the value of $Y$!

On the other hand, integrating over any $\mathcal{F}$-measurable event is unrealistically restrictive. For instance, in the two-coins Example 2 of Level 13, it would say we should be able to integrate over $A = \{(H, H)\}$. But that would tell us to get $\frac{1}{2}$ (on this set we have $X = 2$), and also get $\frac{1}{4}$ (integrating over $\{(H, T)\}$). Our sanity check (ii) says we must make the same guess for both $(H, H)$ and $(H, T)$ because $Y$ is the same on both.

In general, integrating over events not in $\mathcal{G}$ is expecting us to take account of information we don't yet have. Integrating over $\mathcal{G}$-measurable events requires us to take account of all the information we *do* have, but not asking for information we don't have.

### Versions and almost-sure uniqueness

In the examples above there was "one obvious choice" of the conditional expectation. But actually, in Example 3 there were other choices. For example

$$Z' = \begin{cases} Y + \tfrac{1}{2} & \text{if } Y \neq 0.3 \\ 17 & \text{if } Y = 0.3 \end{cases}$$

is also a version of $\mathbb{E}(X|Y)$. Changing the value on a ($\mathcal{G}$-measurable) set of measure zero doesn't affect the integration condition (iii). In more complicated examples, sometimes there isn't "an obvious choice" for the conditional expectation.

However, from Exercise 2 of Level 13, any two versions agree almost surely — so the conditional expectation is unique up to measure-zero modifications.

### The $L^2$ projection viewpoint

Another way of thinking of the conditional expectation $\mathbb{E}(X | \mathcal{G})$ is that it is the "closest" random variable to $X$ which is $\mathcal{G}$-measurable. Here "closest" is defined by the $L_2$-distance $\mathbb{E}((X - Y)^2)$. Of course, this only makes sense if this distance is defined; for example, if $\mathbb{E}(X^2) < \infty$.

It's worth noting that these expectation conditions form a hierarchy: if for some $n \geq 2$ we have $\mathbb{E}(|X|^n) < \infty$, then writing

$$\mathbb{E}(|X|^{n-1}) = \mathbb{E}(|X|^{n-1} \mathbf{1}_{|X| < 1} + |X|^{n-1} \mathbf{1}_{|X| > 1}) \leq \mathbb{E}(\mathbf{1}_{|X| < 1} + |X|^n \mathbf{1}_{|X| > 1}) \leq 1 + \mathbb{E}(|X|^n)$$

we see $\mathbb{E}(|X|^{n-1}) < \infty$ also. However the other direction is easily seen not to hold: for every $n$ there is a counterexample.

### Conditional pmf example (discrete case)

Suppose $X$ and $Y$ are simple random variables, $\mathbb{E}(|X|), \mathbb{E}(|Y|) < \infty$, and write $X = \sum_i \alpha_i \mathbf{1}_{A_i}$ and $Y = \sum_i \beta_i \mathbf{1}_{B_i}$; suppose (without loss of generality) that the $\alpha_i$ are distinct and the $A_i$ pairwise disjoint, and similarly for the $\beta_j$ and $B_j$. Define the **conditional probability mass function** of $X$ given $Y$ by

$$p_{X|Y}(x | y) = \begin{cases} \frac{\mathbb{P}(X = x, Y = y)}{\mathbb{P}(Y = y)} & \text{if } \mathbb{P}(Y = y) > 0 \\ 0 & \text{if } \mathbb{P}(Y = y) = 0 \end{cases}.$$

Then

$$Z = \sum_i \left(\sum_j x_j \, p_{X|Y}(x_j | y_i)\right) \mathbf{1}_{B_i}$$

is a version of $\mathbb{E}(X | Y)$.

### Conditional density example (continuous case)

Given a pair of random variables $X$ and $Y$, the **joint probability density function** is $f_{XY}(x, y)$ a non-negative function on $\mathbb{R}^2$ such that

$$\mathbb{P}\!\left((X, Y) \in A\right) = \int_A f_{XY}(x, y) \, \mathrm{d}(x, y),$$

for all Borel $A \subseteq \mathbb{R}^2$. When it exists, we say $X$ and $Y$ are **jointly continuous**. If $X$ and $Y$ are both continuous (have pdfs $f_X$ and $f_Y$) and independent, then $f_{XY}(x, y) = f_X(x) f_Y(y)$; the "interesting case" is when we don't have independence.

Define the **conditional probability density function** of $X$ with respect to $Y$ to be

$$f_{X|Y}(x | y) = \begin{cases} \frac{f_{XY}(x, y)}{f_Y(y)} & \text{if } f_Y(y) \neq 0 \\ 0 & \text{if } f_Y(y) = 0 \end{cases}.$$

Then $\int_{x=-\infty}^{\infty} x \, f_{X|Y}(x, Y) \, \mathrm{d}x$ is a version of $\mathbb{E}(X | Y)$.

## Key Results

### definition: Conditional expectation (formal)
**Number:** Definition 4
**Plain English:** $\mathbb{E}(X | \mathcal{G})$ is any random variable $Z$ that is (i) integrable, (ii) $\mathcal{G}$-measurable, and (iii) has the same integral as $X$ over every $\mathcal{G}$-measurable set. It is unique up to almost-sure equality.
**Formal:** Let $X$ be a random variable with $\mathbb{E}(|X|) < \infty$ and $\mathcal{G} \subseteq \mathcal{F}$ a sub-$\sigma$-algebra. A version of $\mathbb{E}(X|\mathcal{G})$ is any $Z$ with: (i) $\mathbb{E}(|Z|) < \infty$, (ii) $Z$ is $\mathcal{G}$-measurable, (iii) $\mathbb{E}(Z\mathbf{1}_A) = \mathbb{E}(X\mathbf{1}_A)$ for all $A \in \mathcal{G}$.
**Load-bearing:** yes

### definition: Conditional pmf
**Number:** Example 4
**Plain English:** For simple (discrete) random variables, the conditional expectation is computed by weighting each value of $X$ by its conditional probability given the observed value of $Y$.
**Formal:** $p_{X|Y}(x|y) = \mathbb{P}(X = x, Y = y)/\mathbb{P}(Y = y)$ when $\mathbb{P}(Y = y) > 0$, else $0$. Then $Z = \sum_i \bigl(\sum_j x_j p_{X|Y}(x_j | y_i)\bigr) \mathbf{1}_{B_i}$ is a version of $\mathbb{E}(X|Y)$.
**Load-bearing:** no

### definition: Conditional density
**Number:** Example 5
**Plain English:** For jointly continuous random variables, the conditional density is the joint density divided by the marginal, and the conditional expectation is obtained by integrating $x$ against this conditional density.
**Formal:** $f_{X|Y}(x|y) = f_{XY}(x,y)/f_Y(y)$ when $f_Y(y) \neq 0$, else $0$. Then $\int_{-\infty}^{\infty} x \, f_{X|Y}(x, Y) \, \mathrm{d}x$ is a version of $\mathbb{E}(X|Y)$.
**Load-bearing:** no

## Exercises

### exercise: $L^2$ projection characterization
**Number:** Exercise 3
**Difficulty:** challenge
**Tags:** conditional-expectation, L2-projection, optimization, measurability

**Question:**
Given a random variable $X$ such that $\mathbb{E}(X^2) < \infty$ on $(\Omega, \mathcal{F}, \mathbb{P})$, suppose $\mathcal{G} \subseteq \mathcal{F}$ is a $\sigma$-algebra.

Prove that a random variable $Y$ which is $\mathcal{G}$-measurable and which, among $\mathcal{G}$-measurable random variables, minimises $\mathbb{E}\!\left((Y - X)^2\right)$, is a version of the conditional expectation $\mathbb{E}(X | \mathcal{G})$.

You may assume that a minimiser exists.

**Hint 1:** Check conditions (i) and (ii), then work on (iii)
Condition (ii) ($\mathcal{G}$-measurability) holds by assumption. For (i), since $\mathbb{E}((Y - X)^2) = d < \infty$ we have $\mathbb{E}(|Y - X|) < \infty$ by triangle inequality, so $\mathbb{E}(|Y|) < \infty$. The hard part is (iii): suppose for contradiction there exists $A \in \mathcal{G}$ such that $\mathbb{E}(X \mathbf{1}_A) \neq \mathbb{E}(Y \mathbf{1}_A)$.

**Hint 2:** Perturb $Y$ by $\rho \mathbf{1}_A$
WLOG suppose $\mathbb{E}(X \mathbf{1}_A) = \mathbb{E}(Y \mathbf{1}_A) + \delta$ for some $\delta > 0$. Set $\rho = \delta \mathbb{P}(A)^{-1}$ and compute $\mathbb{E}((X - Y - \rho \mathbf{1}_A)^2)$. Expand and show this is strictly less than $d = \mathbb{E}((X - Y)^2)$, contradicting minimality.

**Solution:**
Let $d = \inf_Y \mathbb{E}((Y - X)^2)$, where $Y$ ranges over $\mathcal{G}$-measurable random variables. This infimum exists and is a real number because the set is non-empty (it contains $\mathbb{E}(X^2)$, taking $Y$ to be the constant 0 random variable), and bounded below by 0. Suppose $Y$ is a minimiser, i.e. $\mathbb{E}((X - Y)^2) = d$. By definition we have (ii). Because $\mathbb{E}(X^2) < \infty$ we have $\mathbb{E}(|X|) < \infty$, and because $\mathbb{E}((Y - X)^2) = d < \infty$, we have $\mathbb{E}(|Y - X|) < \infty$, so by the triangle inequality we conclude $\mathbb{E}(|Y|) < \infty$, giving (i).

Suppose there is some $A \in \mathcal{G}$ such that $\mathbb{E}(X \mathbf{1}_A) \neq \mathbb{E}(Y \mathbf{1}_A)$. Without loss of generality, suppose $\mathbb{E}(X \mathbf{1}_A) = \mathbb{E}(Y \mathbf{1}_A) + \delta$ for some $\delta > 0$. Necessarily we have $\mathbb{P}(A) > 0$, and set $\rho = \delta \mathbb{P}(A)^{-1}$. Then we have

$$\mathbb{E}\!\left((X - Y - \rho \mathbf{1}_A)^2\right) = \mathbb{E}\!\left((X - Y)^2 \mathbf{1}_A\right) - 2\rho \mathbb{E}\!\left((X - Y) \mathbf{1}_A\right) + \rho^2 \mathbb{P}(A) + \mathbb{E}\!\left((X - Y)^2 \mathbf{1}_{A^c}\right)$$
$$= \mathbb{E}\!\left((X - Y)^2\right) - 2\rho \delta \mathbb{P}(A)^{-1} \cdot \mathbb{P}(A) + \rho^2 \mathbb{P}(A)$$

Wait — let us be more careful:

$$\mathbb{E}\!\left((X - Y - \rho \mathbf{1}_A)^2\right) = \mathbb{E}\!\left((X - Y)^2\right) - 2\rho \mathbb{E}\!\left((X - Y)\mathbf{1}_A\right) + \rho^2 \mathbb{P}(A)$$
$$= d - 2\rho\delta + \rho^2 \mathbb{P}(A) = d - 2\frac{\delta}{\mathbb{P}(A)} \cdot \delta + \frac{\delta^2}{\mathbb{P}(A)^2}\mathbb{P}(A) = d - \frac{\delta^2}{\mathbb{P}(A)} < d,$$

which is a contradiction, since $Y + \rho \mathbf{1}_A$ is $\mathcal{G}$-measurable and achieves a smaller value than the minimum $d$. So $Y$ is a version of $\mathbb{E}(X | \mathcal{G})$.

### exercise: Conditional pmf gives a version of CE
**Number:** Exercise 4
**Difficulty:** standard
**Tags:** conditional-expectation, pmf, simple-random-variables, verification

**Question:**
Suppose $X$ and $Y$ are simple random variables, and write $X = \sum_i \alpha_i \mathbf{1}_{A_i}$ and $Y = \sum_i \beta_i \mathbf{1}_{B_i}$; suppose (without loss of generality) that the $\alpha_i$ are distinct and the $A_i$ pairwise disjoint, and similarly for the $\beta_j$ and $B_j$. Define the conditional probability mass function $p_{X|Y}(x|y)$ as in Example 4, and let

$$Z = \sum_i \left(\sum_j x_j \, p_{X|Y}(x_j | y_i)\right) \mathbf{1}_{B_i}.$$

Prove that $Z$ is a version of $\mathbb{E}(X | Y)$.

**Hint 1:** Check $\sigma(Y)$-measurability
Each $B_i$ is in $\sigma(Y)$ by definition. So $Z$ is a simple function built from $\sigma(Y)$-measurable indicators, hence $\sigma(Y)$-measurable.

**Hint 2:** Verify condition (iii) on minimal non-empty sets
It suffices to verify $\mathbb{E}(Z \mathbf{1}_C) = \mathbb{E}(X \mathbf{1}_C)$ for a minimal non-empty set $C$ in $\sigma(Y)$. By disjointness and minimality, $C = B_i$ for some $i$ with $\mathbb{P}(B_i) > 0$. Compute both sides.

**Solution:**
$\sigma(Y)$-measurability (condition (ii)) is trivial: each $B_i$ is in $\sigma(Y)$ by definition.

For condition (i), we calculate $\mathbb{E}(|Z|)$ directly, as a finite sum:

$$\mathbb{E}(|Z|) = \sum_i \left|\sum_j x_j p_{X|Y}(x_j | y_i)\right| \mathbb{P}(B_i) \leq \sum_i \sum_j |x_j| p_{X|Y}(x_j | y_i) \mathbb{P}(B_i) = \sum_j |x_j| \mathbb{P}(X = x_j) = \mathbb{E}(|X|),$$

where we are allowed to swap the order of finite sums, and we use the fact $\sum_i \mathbb{P}(X = x_j, Y = y_i) = \mathbb{P}(X = x_j)$.

Finally, for condition (iii), given $C \in \sigma(Y)$, we may further assume (because the statement is trivial otherwise) that $\mathbb{P}(C) > 0$. We may take $C$ to be a minimal non-empty set in $\sigma(Y)$. Because $B_i \in \sigma(Y)$ and by disjointness and minimality of $C$, we have either $B_i = C$ or $B_i \cap C = \emptyset$. Again without loss of generality there is exactly one $i$. For this fixed $i$, because the event $Y = y_i$ is $C$, we get

$$\mathbb{E}(Z \mathbf{1}_C) = \sum_j x_j p_{X|Y}(x_j | y_i) \cdot \mathbb{P}(C) = \sum_j x_j \frac{\mathbb{P}(X = x_j, Y = y_i)}{\mathbb{P}(Y = y_i)} \mathbb{P}(C) = \sum_j x_j \mathbb{E}(\mathbf{1}_{A_j} \mathbf{1}_C) = \mathbb{E}(X \mathbf{1}_C)$$

as required. By linearity of expectation, the result extends to any $C \in \sigma(Y)$ (which is a disjoint union of minimal sets).

### exercise: Conditional density gives a version of CE
**Number:** Exercise 5
**Difficulty:** challenge
**Tags:** conditional-expectation, density, Fubini, LOTUS, jointly-continuous

**Question:**
Suppose $X$ and $Y$ are jointly continuous random variables with $\mathbb{E}(|X|), \mathbb{E}(|Y|) < \infty$. Define the conditional probability density function $f_{X|Y}(x|y)$ as above. Prove that $\int_{x=-\infty}^{\infty} x \, f_{X|Y}(x, Y) \, \mathrm{d}x$ is a version of $\mathbb{E}(X | Y)$.

**Hint 1:** Verify condition (i) using Fubini's theorem and LOTUS
Compute $\mathbb{E}(|Z|)$ by writing it as $\int_\Omega \left|\int_{-\infty}^{\infty} x f_{X|Y}(x, Y) \, \mathrm{d}x\right| \mathrm{d}\mathbb{P}$. Use the LOTUS to convert $\mathrm{d}\mathbb{P}$ integration to integration over the reals against $f_Y(y)$, then apply Fubini's theorem.

**Hint 2:** For condition (iii), use LOTUS and Fubini
Given $A \in \sigma(Y)$, we have $A = Y^{-1}(B)$ for some Borel $B$. Compute $\mathbb{E}(Z \mathbf{1}_A)$ by integrating: use LOTUS to get an integral over $\mathbb{R}$ against $f_Y(y)$, restrict to $y \in Y(A)$, substitute the definition of $f_{X|Y}$, and recognize the result as $\mathbb{E}(X \mathbf{1}_A)$ via a 2D LOTUS.

**Solution:**
We prove that the given integral $Z = \int_{x=-\infty}^{\infty} x \, f_{X|Y}(x, Y) \, \mathrm{d}x$ satisfies the properties of conditional expectation.

**Condition (i):** We have

$$\mathbb{E}(|Z|) = \int_\Omega \left|\int_{x=-\infty}^{\infty} x f_{X|Y}(x, Y) \, \mathrm{d}x\right| \mathrm{d}\mathbb{P} \leq \int_\Omega \int_{x=-\infty}^{\infty} |x| f_{X|Y}(x, Y) \, \mathrm{d}x \, \mathrm{d}\mathbb{P}.$$

By LOTUS (converting from $\mathrm{d}\mathbb{P}$ to integration over $y$ against $f_Y$):

$$= \int_{x=-\infty}^{\infty} \int_\Omega |x| f_{X|Y}(x, Y) \, \mathrm{d}\mathbb{P} \, \mathrm{d}x = \int_{x=-\infty}^{\infty} \int_{y=-\infty}^{\infty} |x| f_{X|Y}(x, y) f_Y(y) \, \mathrm{d}y \, \mathrm{d}x$$
$$= \int_{x=-\infty}^{\infty} |x| \int_{y=-\infty}^{\infty} f_{XY}(x, y) \, \mathrm{d}y \, \mathrm{d}x = \int_{x=-\infty}^{\infty} |x| f_X(x) \, \mathrm{d}x = \mathbb{E}(|X|) < \infty.$$

**Condition (ii):** For $\sigma(Y)$-measurability, we use the fact (which we did not prove) that $g(y) = \int_{x=-\infty}^{\infty} f_{X|Y}(x, y) \, \mathrm{d}x$ is a measurable function of $y$. Then $g(Y)$ is a $\sigma(Y)$-measurable function as proved in Notes 2.

**Condition (iii):** Given $A \in \sigma(Y)$, we have $A = Y^{-1}(B)$ for some Borel $B \subseteq \mathbb{R}$. Then:

$$\mathbb{E}(\mathbf{1}_A Z) = \int_\Omega \mathbf{1}_A \int_{x=-\infty}^{\infty} x f_{X|Y}(x, Y) \, \mathrm{d}x \, \mathrm{d}\mathbb{P} = \int_{x=-\infty}^{\infty} \int_{y=-\infty}^{\infty} x f_{X|Y}(x, y) \mathbf{1}_{y \in Y(A)} f_Y(y) \, \mathrm{d}y \, \mathrm{d}x$$
$$= \int_{x=-\infty}^{\infty} \int_{y=-\infty}^{\infty} x f_{XY}(x, y) \mathbf{1}_{y \in Y(A)} \, \mathrm{d}y \, \mathrm{d}x = \int_{(x,y) \in \mathbb{R}^2} x f_{XY}(x, y) \mathbf{1}_{y \in Y(A)} \, \mathrm{d}\lambda(x, y).$$

This integral is equal — by a two-dimensional version of the LOTUS — to $\mathbb{E}_{X,Y}(X \mathbf{1}_{Y \in Y(A)}) = \mathbb{E}_{X,Y}(X \mathbf{1}_A) = \mathbb{E}(X \mathbf{1}_A)$ as required. Note here we are using the fact that $A \in \sigma(Y)$ to conclude that $Y(\omega) \in Y(A)$ if and only if $\omega \in A$.
