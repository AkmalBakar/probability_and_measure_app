# Level 6: Measurable Functions & Random Variables

## Reading

This level answers a fundamental question: what exactly *is* a random variable in the general (possibly uncountable) setting? In the discrete world (Level 2), a random variable was simply a function $X : \Omega \to \mathbb{R}$. That was fine because every subset of a finite $\Omega$ was an event. Now that we are working with $\sigma$-algebras and measure spaces, we need to be more careful.

### The problem with arbitrary functions

Suppose we have a probability space $(\Omega, \mathcal{F}, \mathbb{P})$ where $\Omega$ is uncountable. We want to ask questions like "what is the probability that $X \in [a, b]$?" For that question to have an answer, we need the set $X^{-1}([a, b]) = \{\omega \in \Omega : X(\omega) \in [a, b]\}$ to be an event, i.e., to belong to $\mathcal{F}$. An arbitrary function from $\Omega$ to $\mathbb{R}$ might not have this property. So we restrict attention to functions where we *can* ask probabilistic questions — and these are the **measurable functions**.

### Measurable functions and random variables

The definition is cleanly abstract. Let $(Y, \mathcal{F}_Y)$ and $(Z, \mathcal{F}_Z)$ be measurable spaces (sets equipped with $\sigma$-algebras, but no measure specified yet). A function $f : Y \to Z$ is called $(\mathcal{F}_Y, \mathcal{F}_Z)$-**measurable** if:

$$f^{-1}(F) \in \mathcal{F}_Y \quad \text{for all } F \in \mathcal{F}_Z.$$

In words: the preimage of every "measurable set" in the codomain is a "measurable set" in the domain. When the domain carries a probability measure, we call such a function a **random variable**. When $Z = \mathbb{R}$ and $\mathcal{F}_Z = \mathcal{B}(\mathbb{R})$ is the Borel $\sigma$-algebra, we say $f$ is a (real-valued) random variable.

Note: by convention, "random variable" usually means real-valued. We use capital letters $X, Y$ for random variables when doing probability, and $f, g$ when emphasising the function-theoretic perspective.

<details>
<summary>Formal Statement (Definition 1)</summary>

**Definition 1 (Measurable function, random variable).** Let $(Y, \mathcal{F}_Y)$ and $(Z, \mathcal{F}_Z)$ be measurable spaces. A function $f : Y \to Z$ is $(\mathcal{F}_Y, \mathcal{F}_Z)$-measurable if $f^{-1}(F) \in \mathcal{F}_Y$ for all $F \in \mathcal{F}_Z$. If $(Y, \mathcal{F}_Y)$ carries a probability measure, $f$ is a random variable.

</details>

### The generator check (Theorem 2)

Checking $f^{-1}(F) \in \mathcal{F}_Y$ for *every* $F \in \mathcal{F}_Z$ sounds like a lot of work. Fortunately, it is enough to check on a **generating set**. If $\mathcal{G}$ is any collection of subsets of $Z$ with $\sigma(\mathcal{G}) = \mathcal{F}_Z$, and $f^{-1}(G) \in \mathcal{F}_Y$ for all $G \in \mathcal{G}$, then $f$ is $(\mathcal{F}_Y, \mathcal{F}_Z)$-measurable.

The proof defines $\mathcal{F}_Z' = \{F \in \mathcal{F}_Z : f^{-1}(F) \in \mathcal{F}_Y\}$ and observes that $\mathcal{F}_Z'$ is a $\sigma$-algebra (because inverse images commute with complements, unions, and intersections). Since $\mathcal{G} \subseteq \mathcal{F}_Z'$, we get $\mathcal{F}_Z = \sigma(\mathcal{G}) \subseteq \sigma(\mathcal{F}_Z') = \mathcal{F}_Z'$, and hence $\mathcal{F}_Z' = \mathcal{F}_Z$.

**Practical consequence:** To verify that $f : \Omega \to \mathbb{R}$ is a real-valued random variable, it suffices to check that $f^{-1}((-\infty, c]) \in \mathcal{F}$ for all $c \in \mathbb{R}$, since the intervals $(-\infty, c]$ generate $\mathcal{B}(\mathbb{R})$.

<details>
<summary>Formal Statement & Proof (Theorem 2)</summary>

**Theorem 2 (Generator check).** Suppose $f : Y \to Z$ is a function and $(Y, \mathcal{F}_Y)$ and $(Z, \mathcal{F}_Z)$ are measurable spaces. Let $\mathcal{G}$ be any collection of subsets of $Z$ such that $\sigma(\mathcal{G}) = \mathcal{F}_Z$. If $f^{-1}(G) \in \mathcal{F}_Y$ for all $G \in \mathcal{G}$, then $f$ is $(\mathcal{F}_Y, \mathcal{F}_Z)$-measurable.

**Proof.**

1. **Step 1 — Define the "good sets."** Let $\mathcal{F}_Z' = \{F \in \mathcal{F}_Z : f^{-1}(F) \in \mathcal{F}_Y\}$.
2. **Step 2 — Show $\mathcal{F}_Z'$ is a $\sigma$-algebra.** This follows because inverse images commute with all set operations: $f^{-1}(F^c) = (f^{-1}(F))^c \in \mathcal{F}_Y$ if $f^{-1}(F) \in \mathcal{F}_Y$, and $f^{-1}(\bigcup F_n) = \bigcup f^{-1}(F_n) \in \mathcal{F}_Y$ if each $f^{-1}(F_n) \in \mathcal{F}_Y$.
3. **Step 3 — Conclude.** By hypothesis, $\mathcal{G} \subseteq \mathcal{F}_Z'$. Since $\mathcal{F}_Z'$ is a $\sigma$-algebra containing $\mathcal{G}$, we have $\mathcal{F}_Z = \sigma(\mathcal{G}) \subseteq \sigma(\mathcal{F}_Z') = \mathcal{F}_Z'$. But $\mathcal{F}_Z' \subseteq \mathcal{F}_Z$ by definition, so $\mathcal{F}_Z' = \mathcal{F}_Z$.

$\square$

</details>

### The $\sigma$-algebra generated by a random variable

Given a measurable function $f : (Y, \mathcal{F}_Y) \to (Z, \mathcal{F}_Z)$, we define:

$$\sigma(f) = \{f^{-1}(F) : F \in \mathcal{F}_Z\},$$

the $\sigma$-algebra **generated by** $f$. This is a sub-$\sigma$-algebra of $\mathcal{F}_Y$. Intuitively, $\sigma(f)$ captures exactly the **information carried by $f$**: if you know the value of $f(\omega)$, you can determine membership in any set in $\sigma(f)$, and nothing more. This idea is central to conditional expectation (which comes later in the course).

<details>
<summary>Formal Statement (Definition 3)</summary>

**Definition 3 ($\sigma$-algebra generated by a random variable).** Let $f : (Y, \mathcal{F}_Y) \to (Z, \mathcal{F}_Z)$ be measurable. Then $\sigma(f) = \{f^{-1}(F) : F \in \mathcal{F}_Z\}$ is the $\sigma$-algebra generated by $f$, a sub-$\sigma$-algebra of $\mathcal{F}_Y$.

</details>

### Sure, almost sure, almost everywhere

In probability, we often "do not care" about events of probability zero.

- An event $F$ in $(\Omega, \mathcal{F}, \mathbb{P})$ is **sure** if $F = \Omega$, and **almost sure** if $\mathbb{P}(F) = 1$.
- Two random variables $f$ and $g$ are **almost surely equal** (written $f = g$ a.s.) if $\mathbb{P}(\{\omega : f(\omega) \neq g(\omega)\}) = 0$.
- In a general measure space $(X, \mathcal{F}, \mu)$, we say a property holds **almost everywhere** (a.e.) if the set where it fails is a $\mu$-null set.

An important subtlety: whether $f = g$ a.s. depends on the measure. If we switch from $\mathbb{P}$ to a different probability measure $\mathbb{Q}$ on the same $\sigma$-algebra, the null sets can change, and $f$ and $g$ may be $\mathbb{P}$-a.s. equal but not $\mathbb{Q}$-a.s. equal.

<details>
<summary>Formal Statement (Definition 4)</summary>

**Definition 4 (Sure, almost sure, almost everywhere).** An event $F$ in $(\Omega, \mathcal{F}, \mathbb{P})$ is sure if $F = \Omega$ and almost sure if $\mathbb{P}(F) = 1$. Random variables $f$ and $g$ are almost surely equal if $\mathbb{P}(\{\omega : f(\omega) \neq g(\omega)\}) = 0$. In a general measure space $(X, \mathcal{F}, \mu)$, a property holds almost everywhere if it fails only on a $\mu$-null set.

</details>

### Algebra of measurable functions (Theorem 5)

One of the most reassuring results in measure theory: measurable functions are closed under all the standard operations. If $f, f_1, f_2, \ldots$ and $g$ are measurable functions from $(Y, \mathcal{F})$ to $(\mathbb{R}^{\pm\infty}, \mathcal{B}(\mathbb{R}^{\pm\infty}))$, and $\lambda \in \mathbb{R}$, then the following are all measurable (provided they are defined):

$$f + g, \quad fg, \quad \frac{1}{f}, \quad \lambda f, \quad \inf_{n \in \mathbb{N}} f_n, \quad \sup_{n \in \mathbb{N}} f_n, \quad \liminf_{n \geq 1} f_n, \quad \limsup_{n \geq 1} f_n, \quad \lim_{n \to \infty} f_n.$$

Here $\mathbb{R}^{\pm\infty} = \mathbb{R} \cup \{-\infty, +\infty\}$ is the **extended real line**. We need this because the infimum or supremum of a sequence of real-valued functions might be $\pm\infty$. The topology on $\mathbb{R}^{\pm\infty}$ keeps the usual open intervals and adds $[-\infty, x)$ and $(x, +\infty]$ as open sets.

As special cases, $\max(f, g)$, $\min(f, g)$, $|f|$, $f^+ = \max(f, 0)$, and $f^- = \max(-f, 0)$ are all measurable.

The proof of the "algebraic" part uses the fact that continuous functions are Borel-measurable, combined with the observation that $(f, g)$ maps measurably into $\mathbb{R}^2$. The "analytic" part (inf, sup, etc.) is more interesting and is covered in Exercise 1 below.

<details>
<summary>Formal Statement & Proof (Theorem 5)</summary>

**Theorem 5 (Algebra of measurable functions).** Let $f, f_1, f_2, \ldots$ and $g$ be measurable functions from $(Y, \mathcal{F})$ to $(\mathbb{R}^{\pm\infty}, \mathcal{B}(\mathbb{R}^{\pm\infty}))$, and let $\lambda \in \mathbb{R}$. Then (provided they are defined) the following are all measurable: $f + g$, $fg$, $1/f$, $\lambda f$, $\inf_{n \in \mathbb{N}} f_n$, $\sup_{n \in \mathbb{N}} f_n$, $\liminf_{n \geq 1} f_n$, $\limsup_{n \geq 1} f_n$, and $\lim_{n \to \infty} f_n$.

**Proof.**

*Algebraic operations:* A continuous function from $\mathbb{R}^c$ to $\mathbb{R}^d$ is Borel-measurable (the inverse image of every open set is open, hence Borel). The maps $(x,y) \mapsto x+y$, $(x,y) \mapsto xy$, $x \mapsto 1/x$, $x \mapsto \lambda x$ are all continuous. If $f$ and $g$ are measurable from $(Y, \mathcal{F})$ to $(\mathbb{R}^{\pm\infty}, \mathcal{B}(\mathbb{R}^{\pm\infty}))$, then $(f, g)$ maps measurably into $(\mathbb{R}^{\pm\infty})^2$ (check: for any $(a,b) \times (c,d)$, the inverse image is $f^{-1}((a,b)) \cap g^{-1}((c,d)) \in \mathcal{F}$, and these rectangles generate the product $\sigma$-algebra). Composing measurable functions gives measurable functions.

*Infimum:* For any $c > -\infty$, $\{\omega : \inf_{n \in \mathbb{N}} f_n(\omega) \leq c\} = \bigcap_{n \in \mathbb{N}} \bigcup_{m \in \mathbb{N}} \{\omega : f_m(\omega) \leq c + 1/n\}$. Each set $\{f_m \leq c + 1/n\}$ is measurable, so countable unions and intersections preserve measurability. By Theorem 2, $\inf f_n$ is measurable.

*Supremum:* $\sup_{n \in \mathbb{N}} f_n = -\inf_{n \in \mathbb{N}}(-f_n)$, and $-f_n$ is measurable.

*Liminf/limsup/limit:* $\liminf_{n \geq 1} f_n = \sup_{n \in \mathbb{N}} \inf_{k \geq n} f_k$, $\limsup_{n \geq 1} f_n = \inf_{n \in \mathbb{N}} \sup_{k \geq n} f_k$, and $\lim f_n = \liminf f_n$ when the limit exists.

$\square$

</details>

**Caveat on countability:** The inf and sup of a *countable* family of measurable functions are measurable. The sup of an *uncountable* family might not be. For example, $\mathbf{1}_{\{x\}}$ is measurable for each $x \in \mathbb{R}$, and $\mathbf{1}_S = \sup_{x \in S} \mathbf{1}_{\{x\}}$, but $\mathbf{1}_S$ need not be measurable if $S \notin \mathcal{B}(\mathbb{R})$.

### Standard distributions

Now that we have the formal machinery, let us meet the most important random variables. In each case, we specify the **distribution** — the probabilities of the values the random variable takes — rather than the underlying probability space.

**Bernoulli $\text{Ber}(p)$.** A measurable function $X : \Omega \to \{0, 1\}$ with $\mathbb{P}(X = 1) = p$. This is the simplest non-trivial random variable: success or failure. The indicator function $\mathbf{1}_A$ (which is 1 on $A$ and 0 on $A^c$) is a Bernoulli random variable with $p = \mathbb{P}(A)$.

**Binomial $\text{Bin}(n, p)$.** For $n \in \mathbb{N}$ and $p \in [0, 1]$, $X \sim \text{Bin}(n, p)$ means:

$$\mathbb{P}(X = k) = \binom{n}{k} p^k (1 - p)^{n-k}, \quad 0 \leq k \leq n.$$

This counts the number of successes in $n$ independent trials, each with success probability $p$. Equivalently, $X = X_1 + \cdots + X_n$ where each $X_i \sim \text{Ber}(p)$ independently.

**Poisson $\text{Poi}(\lambda)$.** For $\lambda > 0$, $X \sim \text{Poi}(\lambda)$ means:

$$\mathbb{P}(X = k) = e^{-\lambda} \frac{\lambda^k}{k!}, \quad k \geq 0.$$

The Poisson distribution arises as the limit of $\text{Bin}(n, \lambda/n)$ as $n \to \infty$ (Exercise 2). It models the count of rare events over a large number of trials.

**Gaussian $N(\mu, \sigma^2)$.** This is our first *continuous* random variable. $X \sim N(\mu, \sigma^2)$ has density:

$$f_X(x) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\!\left(-\frac{(x - \mu)^2}{2\sigma^2}\right).$$

Probabilities are computed by integration: $\mathbb{P}(X \in [a, b]) = \int_a^b f_X(x) \, dx$. The standard normal $N(0, 1)$ is the most fundamental continuous distribution.

**Exponential $\text{Exp}(\lambda)$.** For $\lambda > 0$, $X \sim \text{Exp}(\lambda)$ takes values in $[0, \infty)$ with density $f_X(x) = \lambda e^{-\lambda x}$ for $x \geq 0$. It models waiting times and has the memoryless property: $\mathbb{P}(X > s + t \mid X > s) = \mathbb{P}(X > t)$.

## Key Results

### definition: Measurable function, random variable
**Number:** Definition 1
**Plain English:** A function between measurable spaces is measurable if pulling back any measurable set in the codomain gives a measurable set in the domain. When the domain carries a probability measure, such a function is called a random variable.
**Formal:** Let $(Y, \mathcal{F}_Y)$ and $(Z, \mathcal{F}_Z)$ be measurable spaces. A function $f : Y \to Z$ is $(\mathcal{F}_Y, \mathcal{F}_Z)$-measurable if $f^{-1}(F) \in \mathcal{F}_Y$ for all $F \in \mathcal{F}_Z$. If $(Y, \mathcal{F}_Y)$ carries a probability measure, $f$ is a random variable. If $Z = \mathbb{R}$ and $\mathcal{F}_Z = \mathcal{B}(\mathbb{R})$, we call $f$ a (real-valued) random variable.
**Load-bearing:** yes

### theorem: Generator check for measurability
**Number:** Theorem 2
**Plain English:** To verify measurability, you don't need to check every set in $\mathcal{F}_Z$ — it suffices to check a generating set. For real-valued functions, checking intervals $(-\infty, c]$ is enough.
**Formal:** Let $f : Y \to Z$ be a function between measurable spaces $(Y, \mathcal{F}_Y)$ and $(Z, \mathcal{F}_Z)$. If $\mathcal{G}$ is a collection of subsets of $Z$ with $\sigma(\mathcal{G}) = \mathcal{F}_Z$ and $f^{-1}(G) \in \mathcal{F}_Y$ for all $G \in \mathcal{G}$, then $f$ is $(\mathcal{F}_Y, \mathcal{F}_Z)$-measurable.
**Proof:** Define $\mathcal{F}_Z' = \{F \in \mathcal{F}_Z : f^{-1}(F) \in \mathcal{F}_Y\}$. Since inverse images commute with set operations ($f^{-1}(F^c) = (f^{-1}(F))^c$ and $f^{-1}(\bigcup F_n) = \bigcup f^{-1}(F_n)$), $\mathcal{F}_Z'$ is a $\sigma$-algebra. By hypothesis $\mathcal{G} \subseteq \mathcal{F}_Z'$, so $\mathcal{F}_Z = \sigma(\mathcal{G}) \subseteq \sigma(\mathcal{F}_Z') = \mathcal{F}_Z'$, and since $\mathcal{F}_Z' \subseteq \mathcal{F}_Z$ by definition, $\mathcal{F}_Z' = \mathcal{F}_Z$. $\square$
**Key technique:** Good sets principle (collect all sets where the desired property holds, show the collection is a $\sigma$-algebra)
**Load-bearing:** yes

### definition: $\sigma$-algebra generated by a random variable
**Number:** Definition 3
**Plain English:** The $\sigma$-algebra generated by $f$ consists of all preimages of measurable sets under $f$. It captures exactly the information that $f$ reveals.
**Formal:** Let $f : (Y, \mathcal{F}_Y) \to (Z, \mathcal{F}_Z)$ be measurable. Then $\sigma(f) = \{f^{-1}(F) : F \in \mathcal{F}_Z\}$ is the $\sigma$-algebra generated by $f$, a sub-$\sigma$-algebra of $\mathcal{F}_Y$.
**Load-bearing:** yes

### definition: Sure, almost sure, almost everywhere
**Number:** Definition 4
**Plain English:** "Almost sure" means probability 1 (we don't care about probability-zero exceptions). Two random variables are a.s. equal if they differ only on a null set.
**Formal:** An event $F$ in $(\Omega, \mathcal{F}, \mathbb{P})$ is sure if $F = \Omega$ and almost sure if $\mathbb{P}(F) = 1$. Random variables $f$ and $g$ are almost surely equal if $\mathbb{P}(\{\omega : f(\omega) \neq g(\omega)\}) = 0$. In a general measure space $(X, \mathcal{F}, \mu)$, a property holds almost everywhere if it fails only on a $\mu$-null set.
**Load-bearing:** yes

### theorem: Algebra of measurable functions
**Number:** Theorem 5
**Plain English:** Measurable functions are closed under arithmetic operations and countable limiting operations. You can add, multiply, take reciprocals, scale, and take countable inf, sup, liminf, limsup, and limits of measurable functions and stay measurable.
**Formal:** Let $f, f_1, f_2, \ldots$ and $g$ be measurable functions from $(Y, \mathcal{F})$ to $(\mathbb{R}^{\pm\infty}, \mathcal{B}(\mathbb{R}^{\pm\infty}))$, and let $\lambda \in \mathbb{R}$. Then (provided they are defined) the following are all measurable: $f + g$, $fg$, $1/f$, $\lambda f$, $\inf_{n \in \mathbb{N}} f_n$, $\sup_{n \in \mathbb{N}} f_n$, $\liminf_{n \geq 1} f_n$, $\limsup_{n \geq 1} f_n$, and $\lim_{n \to \infty} f_n$.
**Proof:** *Algebraic operations:* Continuous functions are Borel-measurable. The maps $(x,y) \mapsto x+y$, $(x,y) \mapsto xy$, $x \mapsto 1/x$, $x \mapsto \lambda x$ are continuous, and $(f,g)$ maps measurably into $(\mathbb{R}^{\pm\infty})^2$ (the inverse image of any rectangle $(a,b) \times (c,d)$ is $f^{-1}((a,b)) \cap g^{-1}((c,d)) \in \mathcal{F}$). Composing measurable functions gives measurable functions. *Infimum:* For $c > -\infty$, $\{\inf f_n \leq c\} = \bigcap_{n \in \mathbb{N}} \bigcup_{m \in \mathbb{N}} \{f_m \leq c + 1/n\}$, a countable intersection of countable unions of measurable sets. By Theorem 2, $\inf f_n$ is measurable. *Supremum:* $\sup f_n = -\inf(-f_n)$. *Liminf:* $\liminf f_n = \sup_n \inf_{k \geq n} f_k$. *Limsup and limit:* follow similarly. $\square$
**Key technique:** Generator check (Theorem 2) + countable set operations
**Load-bearing:** yes

## Exercises

### exercise: Infimum is measurable
**Number:** Exercise 1
**Difficulty:** standard
**Tags:** measurability, infimum, supremum, liminf, limsup, set-equality

**Question:**
Prove that for any $y_1, y_2, \ldots \in \mathbb{R}^{\pm\infty}$ and any $c > -\infty$, we have $\inf_{n \in \mathbb{N}} y_n \leq c$ if and only if

$$\forall n \in \mathbb{N}, \; \exists m \in \mathbb{N} : y_m \leq c + \tfrac{1}{n}.$$

Deduce that for any $c > -\infty$:

$$\left\{\omega : \inf_{n \geq 1} f_n(\omega) \leq c\right\} = \bigcap_{n \in \mathbb{N}} \bigcup_{m \in \mathbb{N}} \left\{\omega : f_m(\omega) \leq c + \tfrac{1}{n}\right\}.$$

Hence show that if $f_1, f_2, \ldots : \Omega \to \mathbb{R}^{\pm\infty}$ are measurable functions, then $\inf_{n \in \mathbb{N}} f_n$ is measurable. Deduce that $\sup_{n \in \mathbb{N}} f_n$, $\liminf_{n \geq 1} f_n$, $\limsup_{n \geq 1} f_n$, and $\lim_{n \to \infty} f_n$ (when it exists) are all measurable. *(Hint: use that the sequence of infima $\inf_{k \geq n} f_k$ is monotone increasing.)*

**Hint 1:** Start with the equivalence for real sequences
By definition, $\inf_{n \in \mathbb{N}} y_n \leq c$ means that $c + \varepsilon$ is not a lower bound of $\{y_1, y_2, \ldots\}$ for any $\varepsilon > 0$. Equivalently, for every $\varepsilon > 0$, some $y_m$ satisfies $y_m < c + \varepsilon$. Why is it enough to check $\varepsilon = 1/n$ for $n \in \mathbb{N}$?

**Hint 2:** Translate the pointwise statement into a set equality
An element $\omega$ is in the left-hand side if and only if the sequence of real numbers $f_1(\omega), f_2(\omega), \ldots$ satisfies the condition from the first part. Use the set equality to write $f^{-1}((-\infty, c])$ as a countable intersection of countable unions of measurable sets. For the deductions, note that $\sup f_n = -\inf(-f_n)$, $\liminf f_n = \sup_n \inf_{k \geq n} f_k$, and $\lim f_n = \liminf f_n$ when the limit exists.

**Solution:**
**The equivalence.** ($\Rightarrow$) If $\inf y_n \leq c$, then for each $n \in \mathbb{N}$, $c + 1/n > c \geq \inf y_n$, so $c + 1/n$ is not a lower bound. Hence there exists $m$ with $y_m < c + 1/n$, which gives $y_m \leq c + 1/n$ (in fact strictly less).

($\Leftarrow$) For each $n$, we have some $y_{m_n} \leq c + 1/n$. Replacing $n$ with $2n$, we get $y_{m_{2n}} \leq c + 1/(2n) < c + 1/n$, so $\inf y_n \leq y_{m_{2n}} < c + 1/n$ for all $n$. Taking $n \to \infty$ gives $\inf y_n \leq c$.

For $c = -\infty$: $\inf y_n = -\infty$ if and only if for all $n \in \mathbb{N}$ there exists $m$ with $y_m < -n$, which is just the definition of infimum being $-\infty$.

**The set equality.** An element $\omega$ belongs to the left-hand side iff $\inf_{n \geq 1} f_n(\omega) \leq c$ iff (by the equivalence above) for all $n \in \mathbb{N}$, there exists $m \in \mathbb{N}$ with $f_m(\omega) \leq c + 1/n$ iff $\omega \in \bigcap_{n \in \mathbb{N}} \bigcup_{m \in \mathbb{N}} \{\omega : f_m(\omega) \leq c + 1/n\}$.

**Measurability of inf.** Let $f = \inf_{n \in \mathbb{N}} f_n$. We use the generating set $\mathcal{G} = \{(-\infty, c] : c \in \mathbb{R}^{\pm\infty}\}$ for $\mathcal{B}(\mathbb{R}^{\pm\infty})$. By the set equality, $f^{-1}((-\infty, c]) = \bigcap_{n} \bigcup_{m} \{\omega : f_m(\omega) \leq c + 1/n\}$. Each set $\{f_m(\omega) \leq c + 1/n\}$ is in $\mathcal{F}$ by measurability of $f_m$. Since $\mathcal{F}$ is closed under countable unions and intersections, $f^{-1}((-\infty, c]) \in \mathcal{F}$. By Theorem 2, $f$ is measurable.

**Deductions.**
- *Supremum:* $\sup_{n} f_n = -\inf_{n}(-f_n)$. Since $-f_n$ is measurable (Theorem 5, algebraic part) and $\inf$ of measurable functions is measurable, so is $\sup$.
- *Liminf:* $\liminf_{n \geq 1} f_n = \sup_{n \in \mathbb{N}} \inf_{k \geq n} f_k$. Each $g_n = \inf_{k \geq n} f_k$ is measurable by the inf result. Then $\sup_n g_n$ is measurable by the sup result.
- *Limsup:* $\limsup_{n \geq 1} f_n = \inf_{n \in \mathbb{N}} \sup_{k \geq n} f_k$, measurable by the same reasoning.
- *Limit:* If $\lim f_n$ exists, then $\lim f_n = \liminf f_n = \limsup f_n$, which is measurable.

### exercise: Binomial to Poisson convergence
**Number:** Exercise 2
**Difficulty:** standard
**Tags:** distributions, Poisson, binomial, convergence, limit

**Question:**
Prove that for any fixed $k \in \mathbb{N}_0$ and $\lambda > 0$, we have

$$\mathbb{P}\!\left(\text{Bin}\!\left(n, \tfrac{\lambda}{n}\right) = k\right) \to \mathbb{P}(\text{Poi}(\lambda) = k)$$

as $n \to \infty$.

**Hint 1:** Write out the binomial probability explicitly
$\mathbb{P}(\text{Bin}(n, \lambda/n) = k) = \binom{n}{k} (\lambda/n)^k (1 - \lambda/n)^{n-k}$. Separate this into a product of several terms, some depending on $n$ and some not.

**Hint 2:** Identify each factor's limit
Rewrite $\binom{n}{k}(\lambda/n)^k = \frac{n(n-1)\cdots(n-k+1)}{k!} \cdot \frac{\lambda^k}{n^k}$. As $n \to \infty$ with $k$ fixed, $\frac{n(n-1)\cdots(n-k+1)}{n^k} \to 1$. Also recall that $(1 - \lambda/n)^n \to e^{-\lambda}$ and $(1 - \lambda/n)^{-k} \to 1$.

**Solution:**
We compute directly:

$$\mathbb{P}\!\left(\text{Bin}\!\left(n, \tfrac{\lambda}{n}\right) = k\right) = \binom{n}{k} \left(\frac{\lambda}{n}\right)^k \left(1 - \frac{\lambda}{n}\right)^{n-k}.$$

Rewrite the binomial coefficient times $(\lambda/n)^k$:

$$\binom{n}{k}\left(\frac{\lambda}{n}\right)^k = \frac{n(n-1)\cdots(n-k+1)}{k!} \cdot \frac{\lambda^k}{n^k} = \frac{\lambda^k}{k!} \cdot \frac{n(n-1)\cdots(n-k+1)}{n^k}.$$

The remaining factor is $(1 - \lambda/n)^{n-k} = (1 - \lambda/n)^n \cdot (1 - \lambda/n)^{-k}$.

Now take $n \to \infty$ with $k$ and $\lambda$ fixed:

1. $\frac{n(n-1)\cdots(n-k+1)}{n^k} = \prod_{j=0}^{k-1}\left(1 - \frac{j}{n}\right) \to 1$
2. $(1 - \lambda/n)^n \to e^{-\lambda}$ (standard limit)
3. $(1 - \lambda/n)^{-k} \to 1$ (since $k$ is fixed)

Multiplying these limits together:

$$\mathbb{P}\!\left(\text{Bin}\!\left(n, \tfrac{\lambda}{n}\right) = k\right) \to \frac{\lambda^k}{k!} \cdot 1 \cdot e^{-\lambda} \cdot 1 = e^{-\lambda}\frac{\lambda^k}{k!} = \mathbb{P}(\text{Poi}(\lambda) = k).$$
